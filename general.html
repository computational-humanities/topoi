Projekte Overview¶

Projects are divided in subfolders. Each project folder contains /data for original data, and /program for usable versions of programs.

Geography Omega¶

Analysis of historical coordinates originating in Ptolemy’s Geography.

Medieval Diagrams¶

Analysis of medieval diagram drawings. Focus is on several typical attributes and their change in time.

To load the modules into a Jupyter notebook use:

import modulename

or:

from modulename import function

medDiaJson.reducedData(dataframe, keyValueList, debug=False)[source]¶

Construct reduced dataframe for (key, value) pair. If value is of type string, partial strings are also matched. Columns are only dropped from dataframe, if all values are NaN. If debug=True: asserts whether values have correct type() for given key. Returns empty dataframe, if no dataset fulfills all conditions.

Parameters:
  • dataframe (pandas.DataFrame) – Name of dataframe for reduction.
  • override (bool) – Override key-value-dtype assertion, default: False
  • keyValueList (list of lists, e.g. [['a','b'],['c','d']]) – List of (key,value) pairs to consider
Returns:

DataFrame
